# Better AI Skills

Collection of AI coding skills installed to enhance development workflows.

## Installation

```bash
# GitHub Release management
npx skills add https://github.com/jezweb/claude-skills --skill github-release

# UI/UX design and prototyping
npx skills add https://github.com/nextlevelbuilder/ui-ux-pro-max-skill --skill ui-ux-pro-max

# Git commit automation with Cloudflare SDK
npx skills add https://github.com/cloudflare/sandbox-sdk --skill git-commit

# Commit and push workflow
npx skills add https://github.com/fradser/dotclaude --skill commit-and-push

# README generation
npx skills add https://github.com/github/awesome-copilot --skill create-readme
```

## Skills Overview

| Skill | Source | Purpose |
|-------|--------|---------|
| **github-release** | [jezweb/claude-skills](https://github.com/jezweb/claude-skills) | Automate GitHub release creation and management |
| **ui-ux-pro-max** | [nextlevelbuilder/ui-ux-pro-max-skill](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill) | UI/UX design patterns, prototyping, and component architecture |
| **git-commit** | [cloudflare/sandbox-sdk](https://github.com/cloudflare/sandbox-sdk) | Structured git commit automation with Cloudflare integration |
| **commit-and-push** | [fradser/dotclaude](https://github.com/fradser/dotclaude) | Streamlined commit and push workflow |
| **create-readme** | [github/awesome-copilot](https://github.com/github/awesome-copilot) | Generate project README files with best practices |
